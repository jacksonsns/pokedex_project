# Introdução

<p>Este é um sistema de catálogo pokemons com Vuejs, que faz comunicação com a Poké API por meio do Axios.</p>

<h2>Recursos:</h2>

* buscar pokemons;
* listar pokemons;
* salve seu pokemon favorito.

# Configuração do projeto

<p>Para compilar e carregar o projeto</p>

```bash

$ npm install

$ npm run server

```
